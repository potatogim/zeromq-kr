**Native core API**

The native core API for ØMQ is [/docs:core-api libzmq]. This is the definitive 'official' API for working with ØMQ, and is mapped into most language bindings.

**Language bindings**

The language bindings (mostly) wrap the native core API, adding more or less abstraction to taste:
